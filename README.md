# Mini-Chat-GPT-LLM-
Mini GPT model using Transformer and LLM along with Retrieval-Augmented Generation (RAG) framework.
🔹 Custom GPT-style transformer model (decoder-only)
🔹 Positional embeddings for sequence understanding
🔹 Wikipedia web scraping pipeline
🔹 RAG (Retrieval-Augmented Generation)
🔹 FAISS-based vector search
🔹 Text generation with temperature sampling
